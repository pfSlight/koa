- list
 > 获取所有ticket列表
 - 参数：无
 - 返回：ticket列表
- lista
 > 获取所有ticket对应的列表名称
 - 参数：无
 - 返回：ticket列表
- del
 > 根据id删除对应的ticket
 - 参数: id
 - 返回：无
- add
 > 添加一个新的ticket
 - 参数： account, phone, password, name, creator
 - 返回：无
- update
 > 根据id修改信息
 - 参数： account, phone, password, name, ticket, id（以上参数可变）
 - 返回：无
- one
 > 根据id查询信息
 - 参数：id
 - 返回：单个对象
