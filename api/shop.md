- list
 > 获取所有shop列表
 - 参数：无
 - 返回：shop列表
- del
 > 根据id删除对应的shop
 - 参数: id
 - 返回：无
- add
 > 添加一个新的shop
 - 参数： account, phone, password, name, address, businessNo
 - 返回：无
- update
 > 根据id修改shop信息
 - 参数： account, phone, password, name, address, businessNo, extraInfo(以上参数可变), id
 - 返回：无
- one
 > 根据单个id查询商家信息
 - 参数： id
 - 返回： 单个shop对象
